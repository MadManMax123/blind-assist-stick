Smart Assist Stick for the Visually Impaired

Smart Stick Visual Overview

Objective

Develop a walking stick that detects nearby obstacles, identifies objects, and provides audio feedback using vibration motors and voice output.

Core Feature Overview

The smart assist stick combines various advanced technologies to improve mobility and safety for visually impaired individuals. Below is an elaboration of the integrated features.

  • Obstacle Detection: Ultrasonic sensors measure distance and activate internal vibration motors with increasing frequency as obstacles approach.
  • Object Recognition: Pi Camera paired with MobileNet SSD enables object identification and classification into useful categories like human, pet, vehicle, etc.
  • Voice Feedback: Recognized objects are announced using pre-recorded MP3s through an AUX-connected audio device.
  • USB-C Charging: Fast, reliable power input via USB-C with easy compatibility.
  • LED Battery Indicator: Five-segment LEDs show current charge status and blink during active charging.
  • Internal Build: Vibration motors are embedded inside a hollow PVC stick, making the design sleek and wire-free externally.
  • Pattern-based Vibration: Custom vibration signals give spatial and urgency-based context:
    • Fast Pulse – Danger close
    • Slow Pulse – Distant object
    • Left/Right Pulse – Directional cue
    • Continuous Buzz – Path fully blocked
  • GPS Tracking: Real-time location monitoring for caregivers via GPS module. Assists in rescue or navigation tracking.
  • Emergency Button: Sends pre-set distress signals and GPS coordinates with one tap. Also triggers visual/audio alarms if needed.
  • Fingerprint Authentication: Ensures only authorized users can operate and access core and emergency features securely.
  • Gyroscope & Fall Detection: Integrated MPU6050 detects sudden falls or tilts and triggers automatic emergency alerts to caregivers.
  • Solar charging: Integrated solar charging capabilies for extra power even during operation.
  • Weather Alerts: Pulls real-time weather data to provide spoken warnings for rain, extreme heat, storms, or other harsh conditions.
  • Step tracking: Has sensors to track physioloical and health stats such as distance, steps, heart rate and calories.

User Manual

  • Powering On: Charge via USB-C and hold the power button for 3 seconds to turn on.
  • Fingerprint Unlock: Place your finger on the scanner to activate the system.
  • Fingerprint Registration: New users can register fingerprints by placing finger 3 times when in setup mode.
  • Navigation Mode: Begin walking. The stick will vibrate based on proximity and direction of obstacles.
  • Audio Mode: Connect earphones to AUX jack. The stick announces recognized objects or environments.
  • Charging: Connect to USB-C port. LED bars indicate charge level (1 to 5). All lights on = fully charged.
  • Emergency Use: Press and hold the emergency button for 3 seconds to trigger alerts and send your GPS location.
  • Shutdown: Hold power button again for 3 seconds to switch off.

Workflow

1. Obstacle Detection

  • Use ultrasonic sensors to detect objects within a set distance.
  • Trigger vibration motors with increasing intensity as the object approaches.
  • Use pattern variation to indicate direction and urgency.

2. Object Detection

  • Capture images with Pi Camera.
  • Use MobileNet SSD to identify objects (e.g., person, dog, car).
  • Match high-confidence objects with pre-defined categories.

3. Voice Output

  • Play corresponding MP3 messages for recognized objects.
  • Use AUX output to speaker or earphones.
  • Prevent repeated or overlapping messages.

4. Emergency Function

  • When emergency button is pressed, alert is sent via Wi-Fi/GSM module.
  • Optionally play alert sound and blink LED indicator.
  • Share GPS location with emergency contact.

5. Fingerprint Access

  • Authenticate the user before enabling main features.
  • Register fingerprints securely using setup mode.
  • Ensure only the registered person can activate emergency or tracking features.

6. Fall Detection

  • Use gyroscope and accelerometer data to detect sudden falls.
  • Trigger emergency alert if a fall is detected.
  • Provide audio feedback to the user about the fall.
  • Log fall incidents for future reference.

7. Charging

  • Use USB-C for charging the device.
  • LED indicators show battery status (1-5 bars).
  • Automatically switch to low-power mode when battery is low.
  • Provide audio feedback when charging is complete.
  • Use a power bank or Li-ion battery pack for extended usage.
  • Solar charging options for outdoor use.

8. GPS and Weather alerts

  • Use GPS module to track location and provide real-time updates.
  • Fetch weather data using API and provide alerts for extreme conditions.
  • Log location data for future reference.
  • Provide audio feedback about the current weather conditions.

9. Audio mode

  • Connect earphones to the AUX jack.
  • Provide audio feedback about recognized objects and obstacles.
  • Prevent overlapping audio messages.
  • Provide volume control options.

10. Step tracking

  • Use sensors to track steps and distance.
  • Provide audio feedback about the number of steps taken.
  • Log health data for future reference.
  • Provide options to set daily step goals.
  • Integrate with health apps for better tracking.

Hardware Requirements

  • Raspberry Pi 4
  • Pi Camera Module v2
  • Ultrasonic Sensors (HC-SR04) × 2-3
  • Vibration Motors × 5–10 (placed inside the hollow stick)
  • L298N Motor Driver or Transistors
  • Speaker or Earphones (AUX)
  • LEDs × 5 (for battery status)
  • USB-C Charging Module
  • Power Bank / Li-ion Battery Pack
  • GPS Module (e.g., NEO-6M)
  • Emergency Push Button Switch
  • Fingerprint Sensor Module (e.g., R307)
  • Gyroscope + Accelerometer Module (e.g., MPU6050)
  • PVC pipe / actual walking stick

Software Requirements

  • Raspberry Pi OS (Lite)
  • Python 3
  • OpenCV
  • TensorFlow Lite / MobileNet
  • pygame or omxplayer
  • gpsd and gpsd-clients (for GPS)
  • Adafruit Fingerprint Library (for sensor)
  • MPU6050 Python library (for fall detection)
  • Weather API integration (e.g., OpenWeatherMap API)

Extended Features & Optional Upgrades

  • AI-powered voice assistant for more interactivity with contextual feedback.
  • Advanced fall detection with timed voice response and auto-alert countdown.
  • Mobile app integration to monitor battery, location, and emergency logs remotely.
  • Real-time cloud dashboard for caregiver access, logs, and map overlays.
  • Night navigation lights using ambient light sensors to enable low-light usability.
  • Smart terrain classification using pressure and tilt sensors to detect steps, slopes, and water patches.
